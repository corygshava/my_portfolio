My skills my main toolbox
the one highlighted is my proficiency level
HTML
HTML provides the foundational structure of web pages by defining elements displayed on the page. It is essential for all web development and works in tandem with CSS and JavaScript to create functional and visually appealing websites.
CSS
CSS enables developers to separate content from design, allowing for responsive, visually engaging user interfaces. Features like Flexbox, Grid, animations, and media queries empower modern web styling across devices.
Proficient
Javascript
JavaScript brings interactivity to websites by manipulating the DOM, handling events, and enabling asynchronous communication (AJAX). It powers dynamic features like form validation, animations, and real-time updates, and is also widely used on the server-side via Node.js.
PHP
PHP is used to generate dynamic page content, handle form submissions, manage sessions, and interact with databases. It integrates seamlessly with HTML and powers popular platforms like WordPress, making it a staple for backend web development.
SQL
SQL is used to query, update, and manage data stored in database systems like MySQL, PostgreSQL, and SQL Server. It allows developers to retrieve insights, build reports, and maintain structured data efficiently.
secondary
Nodejs
Node.js enables developers to use JavaScript on the server side, allowing for full-stack development using a single language. It's particularly well-suited for building APIs, real-time applications, and microservices due to its non-blocking, event-driven architecture.
react
Developed and maintained by Facebook, React allows developers to build reusable UI components and manage complex application states efficiently using a virtual DOM. It is widely used for creating fast, interactive front-end experiences.
System design
System design ensures that applications are scalable, maintainable, and performant. It involves understanding requirements, selecting appropriate technologies, designing databases, and planning interactions between different parts of a system to meet both current and future needs.
Wordpress
WordPress provides a user-friendly interface for creating blogs, business sites, portfolios, and e-commerce platforms. It supports themes for design customization and plugins for extended functionality, making it a popular choice for both developers and non-technical users.
